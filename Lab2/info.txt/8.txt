Problem 208: 72789. 2Power
For a given number N, print out all the integer powers of two that do not exceed N, in increasing order.

Input format
Given a positive integer N (1 ≤ N ≤ 100).

Output format
Output the answer for the problem.

Examples
Input
50
Output
1 2 4 8 16 32 

Input
32
Output
1 2 4 8 16 32 

Input
1
Output
1 

Input
28
Output
1 2 4 8 16 

Input
100
Output
1 2 4 8 16 32 64
