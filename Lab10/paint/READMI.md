# Paint

## Description
This is a simple Paint application built with Pygame. The user can draw on the canvas using different tools and colors.

## Controls
- B — Brush
- R — Rectangle
- C — Circle
- E — Eraser

## Colors
- 1 — Black
- 2 — Red
- 3 — Green
- 4 — Blue
- 5 — Yellow

## Features
- Free drawing with brush
- Drawing rectangles
- Drawing circles
- Eraser tool
- Color selection
- Simple user interface

## Run
```bash
python main.py
