# Practice 10 - Pygame Projects

## Description
This practice contains three Pygame projects based on extending tutorial examples. Each project demonstrates different game mechanics and user interaction.

---

## Projects

### 1. Racer
A simple racing game where the player controls a car, avoids enemies, and collects coins.

**Features:**
- Player movement (left/right)
- Moving enemy cars
- Random coin spawning
- Coin collection system
- Score display (top-right)

---

### 2. Snake
A classic Snake game with additional mechanics such as levels and increasing speed.

**Features:**
- Snake movement with keyboard
- Collision detection (walls and itself)
- Random food spawning (not on snake)
- Score system
- Level system
- Increasing speed per level

---

### 3. Paint
A simple drawing application with multiple tools and color options.

**Features:**
- Brush drawing
- Rectangle tool
- Circle tool
- Eraser
- Color selection
- Clean UI with canvas separation

---

## Controls Summary

### Racer
- LEFT / RIGHT — move

### Snake
- Arrow keys — move snake

### Paint
- B — Brush  
- R — Rectangle  
- C — Circle  
- E — Eraser  

Colors:
- 1 — Black  
- 2 — Red  
- 3 — Green  
- 4 — Blue  
- 5 — Yellow
