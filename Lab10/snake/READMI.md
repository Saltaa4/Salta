# Snake Game

## Description
This is a classic Snake game built with Pygame. The player controls a snake that grows by eating food while avoiding collisions.

## Controls
- Arrow keys (↑ ↓ ← →) — move the snake

## Features
- Snake movement with keyboard control
- Collision detection (walls and itself)
- Random food spawning (not on snake)
- Score system
- Level system
- Increasing speed with levels

## Game Rules
- Eat food to increase your score
- Every 3 points increases the level
- Speed increases with each level
- Game ends if the snake hits the wall or itself

## Run
```bash
python main.py
