# Racer Game

## Description
This is a simple racing game built with Pygame. The player controls a car and collects coins while avoiding enemies.

## Controls
- LEFT / RIGHT arrows — move the car

## Features
- Moving enemy cars
- Randomly appearing coins
- Coin collection system
- Score displayed in the top right corner

## Objective
Collect as many coins as possible without crashing into enemies.

## Run
```bash
python main.py
