# Music Player

## Description
This is a simple music player built with Pygame. It allows playing and switching between audio tracks using keyboard controls.

## Controls
- P — Play
- S — Stop
- N — Next track
- B — Previous track
- Q — Quit

## Features
- Play/Stop music
- Switch between tracks
- Displays current track name

## Requirements
Audio files must be placed in the `music/` folder.

## Run
```bash
python main.py
