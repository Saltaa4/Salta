# Paint

## Description
This is an upgraded Paint application built with Pygame. It supports drawing basic and geometric shapes, erasing, and color selection.

## Controls
- B — Brush
- R — Rectangle
- C — Circle
- E — Eraser
- S — Square
- T — Right Triangle
- Q — Equilateral Triangle
- H — Rhombus

## Colors
- 1 — Black
- 2 — Red
- 3 — Green
- 4 — Blue
- 5 — Yellow

## Features
- Brush drawing
- Rectangle tool
- Circle tool
- Square tool
- Right triangle tool
- Equilateral triangle tool
- Rhombus tool
- Eraser
- Color selection
- Commented code

## Run
```bash
python main.py
