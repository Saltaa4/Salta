# Practice 11 - Advanced Pygame Projects

## Description
This practice extends previous Pygame projects by adding more advanced game mechanics and features. Each project builds on the implementation from Practice 10.

## Projects

### 1. Racer
A racing game where the player collects coins with different weights while avoiding enemies.

**Features:**
- Player movement (left/right)
- Enemy movement
- Coins with different weights (1, 2, 3)
- Score increases based on coin weight
- Enemy speed increases after collecting N coins
- Commented code

### 2. Snake
A Snake game with advanced food mechanics.

**Features:**
- Snake movement with keyboard
- Collision detection (walls and itself)
- Food with different weights
- Food disappears after a limited time
- Score increases based on food weight
- Commented code

### 3. Paint
A drawing application with multiple geometric tools.

**Features:**
- Brush drawing
- Rectangle and circle tools
- Square tool
- Right triangle tool
- Equilateral triangle tool
- Rhombus tool
- Eraser
- Color selection
- Clean UI with canvas separation
- Commented code

## Controls Summary

### Racer
- LEFT / RIGHT — move car

### Snake
- Arrow keys — move snake

### Paint

Tools:
- B — Brush  
- R — Rectangle  
- C — Circle  
- E — Eraser  
- S — Square  
- T — Right Triangle  
- Q — Equilateral Triangle  
- H — Rhombus  

Colors:
- 1 — Black  
- 2 — Red  
- 3 — Green  
- 4 — Blue  
- 5 — Yellow
