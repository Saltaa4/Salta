# Snake Game

## Description
This is an upgraded Snake game built with Pygame. Food now has different weights and disappears after a limited time.

## Controls
- Arrow keys — move the snake

## Features
- Snake movement with keyboard control
- Collision detection (walls and itself)
- Random food generation
- Food with different weights
- Food timer (food disappears after some time)
- Score display
- Commented code

## Run
```bash
python main.py
